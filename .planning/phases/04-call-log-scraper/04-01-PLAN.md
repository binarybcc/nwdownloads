**Constructor:** Accept `string $username, string $password`. Initialize cURL handle with:

**`public function login(): bool`** — Three-step auth flow:

1. `GET {$this->baseUrl}/Login/` (get cookies)
2. `POST {$this->baseUrl}/servlet/Login` with form data: `EnteredUserID`, `UserID`, `Password` (from constructor), `domain` (empty string)
3. `GET {$this->httpBase}/Common/folder_contents.jsp?menuId=1` (establish HTTP session)
4. Return `str_contains($loginResponse, 'folder_contents.jsp')` — checks step 2 response

**`public function logout(): void`** — `GET {$this->httpBase}/servlet/Logout`

**`public function getCallLogs(string $userKey, string $type): array`** where `$type` is 'placed', 'received', or 'missed':

1. Set user context: `GET {$this->httpBase}/Group/Members/Modify/index.jsp?key={URL-encoded key}` — use `str_replace('::', '%3A%3A', $userKey)` for encoding
2. Fetch call log page: placed = `/User/BasicCallLogs/`, received = `/User/BasicCallLogs/index.jsp?type=1`, missed = `/User/BasicCallLogs/index.jsp?type=2`
3. Call `$this->parseCallLogs($html)` to extract entries
4. If `hasContent($html)` is true AND entries is empty, throw `\RuntimeException('HTML content found but no call log entries parsed for {$type} — possible HTML structure change')`
5. Return array of entries, each: `['name' => string, 'phone' => string, 'datetime' => string]`

**`private function parseCallLogs(string $html): array`**:

1. Use regex: `preg_match_all('/<\/td>\s*<td\s*>([^<\n]+)/', $html, $matches)`
2. Clean values: `array_map(fn($v) => trim(html_entity_decode($v, ENT_QUOTES | ENT_HTML5)), $matches[1])`
3. Filter empty values with `array_values(array_filter(...))`
4. Group into chunks of 3 (name, phone, datetime)
5. Validate each chunk: datetime field must match `'/\d{1,2}\/\d{1,2}\/\d{2}/'`
6. Return array of validated entries
