Key details:

- **ProgramArguments:** SSH to NAS, run PHP 8.2 binary with `fetch_call_logs.php` -- same pattern as auto-import but different script
- **StartCalendarInterval:** Single dict with only `Minute` key set to 5. Per Apple launchd docs, a dict with only Minute fires once per hour at that minute, every hour, every day. The PHP business-hours guard in fetch_call_logs.php handles the 8am-8pm window. No need for 12 separate dict entries.
- **Log paths:** `~/Library/Logs/circulation/call-scraper.log` per CONTEXT.md decision (same directory as auto-import.log)
