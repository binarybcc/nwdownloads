## Color

### Phase-Specific: 8-Bar Gradient (red to green)

This is the primary visual contract for this phase. The gradient communicates urgency: red = past due (act now), green = far out (safe).

| Bar Label | RGBA Value                 | Tailwind Reference | Semantic Meaning               |
| --------- | -------------------------- | ------------------ | ------------------------------ |
| Past Due  | `rgba(239, 68, 68, 0.8)`   | red-500            | Expired — immediate action     |
| This Week | `rgba(245, 158, 11, 0.8)`  | amber-500          | Expiring now — urgent          |
| Next Week | `rgba(251, 191, 36, 0.8)`  | amber-400          | Expiring soon — high priority  |
| Week +2   | `rgba(253, 224, 71, 0.8)`  | yellow-300         | Upcoming — plan outreach       |
| Week +3   | `rgba(250, 240, 137, 0.8)` | yellow-200         | Moderate — schedule contact    |
| Week +4   | `rgba(190, 242, 100, 0.8)` | lime-300           | Comfortable — advance planning |
| Week +5   | `rgba(134, 239, 128, 0.8)` | green-300          | Safe — future renewal window   |
| Week +6   | `rgba(74, 222, 128, 0.8)`  | green-400          | Safe — furthest out            |
| Fallback  | `rgba(156, 163, 175, 0.8)` | gray-400           | Unknown/unexpected label       |

**Color rationale:** All values are from the Tailwind default palette at 0.8 opacity, consistent with the existing 4-bar implementation. The gradient uses 8 visually distinct stops across the red-amber-yellow-lime-green spectrum.

**Important:** The existing "This Week" color changes from `rgba(251, 146, 60, 0.8)` (orange-400) to `rgba(245, 158, 11, 0.8)` (amber-500) to create smoother gradient spacing across 8 stops instead of 4.

## Interaction Contract

### Bar Chart Interactions (all existing, extended to 8 bars)

| Interaction              | Behavior                                     | Change in This Phase                                              |
| ------------------------ | -------------------------------------------- | ----------------------------------------------------------------- |
| Hover bar                | Tooltip shows "{count} subscriptions"        | No change — works for any bar count                               |
| Right-click bar          | Context menu appears with "View subscribers" | No change — reads `chart.data.labels[index]` dynamically          |
| Click "View subscribers" | Subscriber table panel slides in from right  | No change — `getExpirationSubscribers()` needs 4 new switch cases |
| Escape key               | Closes context menu or subscriber panel      | No change                                                         |

### States

| State          | Appearance                                      |
| -------------- | ----------------------------------------------- |
| Loading        | Existing spinner/skeleton (no change)           |
| Data present   | 8 bars rendered with gradient colors            |
| Zero count bar | Bar renders at height 0, still right-clickable  |
| No data        | Chart area empty (existing behavior, no change) |
